Testing en Angular

Karma

Karma es una herramienta que invoca un web server el cual ejecuta el código fuente contra el código de test para cada navegador configurado.

Karma escucha los cambios de los archivos especificados en el archivo de configuración, y por cada cambio, dispara la ejecución de los tests mandando una señal al servidor de testing, y así avisar a todos los navegadores de ejecutar nuevamente el código.

Los navegadores cargar el código fuente en un IFrame, ejecuta los tests y reportan los resultados al servidor.

El servidor recolecta los resultados de todos los navegadores y se los presenta al desarrollador, indicando si pasaron o fallaron los tests.

Jasmine

Jasmine es un framework de testing para Javascript, basado en la idea de BDD (Behaviour-Driven Development o Desarrollo dirigido por comportamientos), el cual combina las técnicas generales y principios del TDD con el DDD (Domain-driven Design o Diseño dirigido por el dominio) y el OOAD (Object-oriented Analysis and Design o Análisis y diseño orientado a objetos).
